CeGCC - Software compatibility list

This page gives an overview of software that works with CeGCC. At the bottom, there is a separate table that lists software that does not and will not work with CeGCC.

Information on this page :

  • Software that is confirmed to work with CeGCC
  • To be rated : does this software work with CeGCC ?
  • Software that does not, and never will work with CeGCC

Please help us to make this list more accurate. Send input to cegcc-devel@lists.sourceforge.net if you're subscribed to that list, or to the list maintainers (cegcc-devel-owner@ ..) otherwise.
